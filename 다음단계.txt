1️⃣ GitHub에서 새 저장소 만들기
───────────────────────────────
   1. https://github.com/new 접속
   2. Repository name: bitcoin-analysis
   3. ⚠️ Public 선택 (중요!)
   4. ❌ 아무것도 체크하지 마세요!
   5. "Create repository" 클릭


2️⃣ GitHub에 올리기
───────────────────────────────
   GitHub에서 보여주는 명령어 중 아래 부분을 복사해서 실행:
   
   ┌─────────────────────────────────────────────────────────┐
   │ git remote add origin https://github.com/사용자명/bitcoin-analysis.git │
   │ git branch -M main                                       │
   │ git push -u origin main                                  │
   └─────────────────────────────────────────────────────────┘
   
   ⚠️ 위 명령어에서 "사용자명"을 본인 GitHub 사용자명으로 변경!


3️⃣ GitHub Actions 실행 (첫 실행)
───────────────────────────────
   1. GitHub 저장소 페이지 → "Actions" 탭
   2. "비트코인 분석 리포트 자동 배포" 클릭
   3. 우측 "Run workflow" 버튼 → "Run workflow" 클릭
   4. 실행 완료까지 대기 (약 1-2분)
   5. 초록색 체크 ✅ 확인


4️⃣ GitHub Pages 활성화
───────────────────────────────
   1. GitHub 저장소 → "Settings" 탭
   2. 왼쪽 메뉴 → "Pages" 클릭
   3. Source:
      - Branch: gh-pages
      - Folder: / (root)
   4. "Save" 클릭
   5. 1-2분 대기


5️⃣ 웹사이트 확인! 🎉
───────────────────────────────
   주소: https://사용자명.github.io/bitcoin-analysis/
   
   예시: https://johndoe.github.io/bitcoin-analysis/
